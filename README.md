# terraform-practice
Pragmatic Terraform on AWS の写経

## Usage
### Install
```
brew install tfenv
tfenv --version
```

```
tfenv install
```

### Setup

```
export GITHUB_TOKEN=12345678901234567890
```

```
terraform init
```

### Dry Run

```
terraform plan
```

### Run

```
terraform apply
```

### Delete

```
terraform destroy
```
